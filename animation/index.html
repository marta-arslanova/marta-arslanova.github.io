Кейфреймы

CSS может превратить элемент веб-страницы в произведение искусства, но такой элемент нельзя назвать интерактивным, пока он не отслеживает события в браузере.

Прелесть кейфрейма познаётся при использовании JavaScript, когда можно включить анимацию в нужный момент или запустить по какому-нибудь случаю.
